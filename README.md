# Plataforma-De-Cargas

Criei este projeto em uma aula da escola ainda, foi meu primeiro projeto

## Sobre
- Esse projeto consiste em um site onde você como motorista, pode procurar facilmente por cargas na web, facilitando o contato com empresas.
- Já como uma empresa, você pode postar suas cargas para que motoristas procurem elas

## Ainda em Desenvolvimento
Isso não passa de um projeto que talvez possa evoluir para uma startup mais para frente. Com um visual simples e facil para o uso de motoristas mais velhos que não tem muita experiência com tecnologia, já que mais de 28% dos motoristas tem mais de 46 anos
